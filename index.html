🧠 Algorithm Explanation

Encryption Process

For each byte in the plaintext:

  1. Convert byte to mapped value m
  2. Generate cryptographically secure random number k
  3. Compute a = gk mod p
  4. Compute b = m × gm×k mod p
  5. The ciphertext is the pair (a, b)
Ciphertext = (gk mod p, m × gm×k mod p)

Decryption Process

For each ciphertext pair (a, b):

  1. For each possible mapped value m in mapping:
  2. Compute am mod p
  3. Check if b ≡ m × am mod p
  4. When match is found, convert m back to byte
For each m in mapping: check if b ≡ m × am mod p
